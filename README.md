# Nature Nest Global 🌿

Nature Nest Global is a premium, fully-responsive static website designed for an Indian food export business. It showcases high-quality agricultural exports such as Foxnuts, Psyllium Husk, Millets, Turmeric, Basmati Rice, and Specialty Coffee to global buyers.

## 🚀 Features

- **Modern & Premium Design**: Features elegant typography (Cormorant Garamond & Outfit fonts), a harmonious color palette (greens, gold, and off-white), and micro-interactions for a luxurious feel.
- **Fully Responsive**: Carefully optimized for seamless viewing across desktop, tablet, and mobile devices without horizontal scrolling issues.
- **Sticky Navigation**: A smooth, always-accessible navigation bar.
- **Floating WhatsApp Integration**: Features a continuous pulse animation to draw attention without being intrusive.
- **Inquiry Form**: A styled, robust contact form tailored for international export queries.
- **Performance Optimized**: Uses lightweight standard web technologies for incredibly fast load times.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for clean structure and SEO.
- **CSS3 (Vanilla)**: Custom styling using CSS variables, flexbox, and CSS Grid for complex layouts like the hero section, product cards, and footers.
- **JavaScript (Vanilla)**: Minimal scripting for intersection observers (scroll reveal animations), form handling, and mobile menu toggles.
- **Bootstrap 5.3.3**: Selectively used for base resetting and utility classes.
- **Lucide Icons**: Crisp, scalable vector icons used throughout the interface.

## 📦 Getting Started

This is a static website, so no complex build processes or servers are required!

1. Clone the repository:
   ```bash
   git clone https://github.com/Palash-r26/Nature_Interview.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Nature_Interview
   ```
3. Open `index.html` in your favorite web browser.

## 🎨 File Structure

- `index.html`: The main structural entry point of the application.
- `Style.css`: Contains all custom styling, CSS variables, animations, and responsive media queries.
- `script.js`: Handles interactive elements like the mobile menu and scroll animations.
- `/photo`: Contains all optimized image assets.
- `/Goldplay`: Custom fonts folder.
